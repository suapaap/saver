• set 안의 중복데이터는 허용하지 않음 : 중복 검사 후 존재하는 최초 데이터만 저장
  • Set() : 파라미터는 배열을 주로 사용함 => 배열 index와 값을 활용하여 집합데이터를 만듬
  • set의 전용 메서드들이 존재함
  • add : 값을 추가 : setName.add(값)
  • size : 크기를 알려줌 : setName.size; //size() X
  • has : 값이 있는지 검사 : setName.has(값) 있으면 true, false
  • delete : 값을 삭제 : setName.delete(값) 성공하면 true, false 반환
  • clear : 모두 삭제 : setName.clear()
  • forEach(v,i,self(set)) : indext == value (Set은 키가 별도로 존재하지 않음)
  • const a = new Set([1,2,3]); : 배열 => Set
  • const b = [...a] : Set => 배열
  • setName.keys(), setName.values(), setName.entries() : 리턴타입 SetIterator : forEach 순회X
