객체 : 식별할 수 있는 것을 상세 특징으로 표현하려한 형식(코드 타입)

  • const로 선언하는 것이 일반적 : 객체의 지속성을 확보하기 위함
  • {}; 속성명: 속성값 ... 으로 표현
  • 속성명은 같은 객체 내에서는 중복되면 안됨
  • 객체 내부의 속성값으로 사용할 수 있는 데이터 타입은 제한이 없다.
  • 객체 속성값으로 익명함수를 정의하면 그 함수를 메서드라고 부른다.
    메서드 명은 속성명이 된다.
// 내가 좋아하는 가수의 객체 이름, 나이, playlist 3가지곡, function() // 노래 3곡을 입력받아(파라미터로 입력) 출력
