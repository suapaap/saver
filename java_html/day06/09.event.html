이벤트 객체는 전역객체 모든 태그에 객체를 참조시킬 수 있다.

  • a태그와 button태그에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할 수 있게 만드는 구조가 일반적
  • a태그는 href속성에 의해 자체 이동(동기화방식)이 구현되어 있어서 별도의 이동속성을 막는 코드가 필요
  • html 태그는 html Element, 속성 attribute, 일반 텍스트 textNode라고 함.
  • html로 출력하기 : innerHTML를 사용
  • text로 출력하기 : innerText를 사용
  • id값으로 객체를 가져올 때 : getElementById('name')
  • css선택자를 사용하여 가져오기 : querySelector('selector')
