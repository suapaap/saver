document 영역에서 사용되어지는 이벤트들

마우스, 키보드, 실행 등의 이벤트의 종류가 매우 많기 때문에 웹에서 자주 사용되는 이벤트를 우선으로 학습하길 권장

  • 마우스 이벤트 : click(), focus(), blur(), input(), change(), mouserover(), mouseout(), select(), scroll(), drag() => drop() 과정 등등.
  • 키보드 이벤트 : keyDown(), keypress(), keyup()
  • 기타 이벤트 관련 메서드 : target(), submit(), reset(), search()
  • 메서드를 태그 속성이나 엘리먼트에 직접 부여하는 방식 이외 addEventListener('eventName', callback function) 메서드를 사용하는 것을 권장
  • addEventListener를 권장하는 이유 : 같은 엘리먼츠에 여러 이벤트를 중복해서 부여할 수 있거, 적용된 이벤트를 선택적으로 제거 가능
